# AI 开发文档索引

欢迎使用 PTab 项目的 AI 开发文档！本文档集旨在帮助 AI 助手和开发者快速理解和参与项目开发。

---

## 必须遵守的原则

所有文档中不得使用emoji

任务完成后必须进行的操作：

- 检查项目是否存在冗余文件、模块、包体、目录，如果有，删除它们
- 将本次更改写入相应的 AI 开发文档中
- 进行日志记录：在日志文件中

## 快速导航

### 新手入门

- 先阅读 [项目概述](overview.md) 了解项目基本情况
- 再查看 [架构设计](architecture.md) 理解整体结构
- 最后根据需要查阅 [组件文档](components.md)

### 开发任务

- 修改 UI 样式 → [样式指南](styles.md)
- 添加新功能 → [架构设计](architecture.md) + [组件文档](components.md)
- 构建和部署 → [开发指南](development.md)
